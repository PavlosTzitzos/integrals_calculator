What is Integral Calculator?

Integral Calculator is an open-source project of a symbolic calculator for integrals. You can compute almost any integral difinite and indifinite:

  • Double Integrals
  • Triple Integrals
  • Line Integrals
    • Scalar Function
    • Vector Function
  • Surface Integrals
  • General Integrals * not ready yet
with all the steps. Also supports two languages : English and Greek.

This is a project that was initially created in MATLAB and then moved to Python with sympy. Then we started adding GUI with tkinter and then then current version is using PySimpleGUI. We are currently working on a C# version using AngouriMath !!

It is free to use even in commercial projects. We work on it, so your requests on issues are likely to be considered within a few days. The project is developed by Pavlos Tzitzos and Vasilis Navrozidis .
