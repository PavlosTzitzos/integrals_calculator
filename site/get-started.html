Getting Started

Head to project's repository were you will find the integralcalculator.py file. You can download it and then build the executable file. To do so follow these steps:

  • Prerequesites: python , sympy , pysimplegui and pyinstaller
  • Then follow the instructions here
You can do the same for the files that are seperated ( ie only double integrals , only triple integrals etc)

The MATLAB files are ready to run.

  • Download one or more files you want or copy the code.
  • Open MATLAB and open the editor ( >>> edit )
  • Paste the code or head to the directory you downloaded the code and open the files
  • Follow instructions in the file.
